## 🔑 Key Concepts

### Room Codes
- 6-character alphanumeric codes (e.g., "A3X9K2")
- Generated by `utils/roomCodeGenerator.js`
- Used to join sessions

### Data Flow
1. User creates session → Firebase
2. Gets room code → Share with team
3. Upload CSV → Parse tasks → Firebase
4. Drag tasks → Update Firebase → All users see change
5. Complete session → Open Jira tabs

### Firebase Structure
```
sessions/{roomCode}/
  ├── metadata/
  ├── participants/
  ├── tasks/
  ├── columns/
  ├── chat/
  ├── currentTurn
  └── turnOrder
```

## 🔧 Utility Functions

### Generate Room Code
```javascript
import { generateRoomCode } from './utils/roomCodeGenerator';
const code = generateRoomCode(); // Returns "A3X9K2"
```

### Parse CSV
```javascript
import { parseJiraCSV } from './utils/csvParser';
const result = await parseJiraCSV(file);
// Returns: { tasks, jiraBaseUrl, totalCount }
```

### Build Jira URL
```javascript
import { buildJiraUrl } from './utils/jiraUrlBuilder';
const url = buildJiraUrl(baseUrl, 'PROJ-123');
```

## 🐛 Debugging Tips

### Firebase Issues
- Check Firebase console → Realtime Database → Data
- Verify .env.local credentials
- Check browser console for errors
- Use Firebase security rules simulator

### React Issues
- Use React DevTools extension
- Check component state in DevTools
- Look for dependency warnings in console

### Network Issues
- Open Network tab in DevTools
- Look for failed Firebase calls
- Check WebSocket connections
